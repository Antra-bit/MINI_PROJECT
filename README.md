
# Python Mini Projects</br>
Overview

This repository contains a collection of beginner-friendly Python mini projects developed as part of my learning journey. These projects were created to practice and strengthen core programming concepts, improve problem-solving abilities, and gain hands-on experience with Python through practical applications.</br>

Projects Included
🧮 Calculator & Converter

A Python-based application that performs arithmetic calculations and simple conversions, showcasing the use of functions, user interaction, and decision-making logic.

⚡ Harry Potter Quiz Game

An interactive quiz application that evaluates user responses through loops and conditional statements while providing an engaging question-and-answer experience.

💰 Expense Tracker

A simple expense management system that enables users to record and monitor expenses, demonstrating fundamental data handling and organization techniques.

🎮 Word Guessing Game

A console-based word game that challenges users to guess a hidden word, highlighting the implementation of loops, conditions, and string manipulation.

Purpose

The primary goal of these projects is to reinforce fundamental Python concepts such as loops, conditional statements, functions, user input handling, and basic data management while building confidence in writing clean and logical code.</br>
